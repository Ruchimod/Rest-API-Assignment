File Contents:

Read_Excel_File.java: Reading Data from Excel File and storing it in Hashmap
post_pet_details.java: For reading pet details from Hashmap and passing them in Response Body of Pet_API
get_pet_details.java: For getting pet details present in Pet_API for a particular Pet_ID.
delete_pet.java: For deleting pet details present in Pet_API for a particular Pet_ID.
Rest_API_Assignment.xml: Contains three tests (Post/Get/Put) in a Suite. We pass parameters to the different Java files from here and execute it for Validation of scenarios.
